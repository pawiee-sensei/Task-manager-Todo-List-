## 📝 Task Manager (Todo List)

A lightweight and scalable **Task Manager** built with **vanilla JavaScript**, focusing on clean architecture and real-world frontend patterns.

---

## 🚀 Features

- Add tasks with validation  
- Delete tasks dynamically  
- Empty state handling  
- Efficient DOM rendering using `DocumentFragment`  
- Event delegation for scalable event handling  
- Input sanitization to prevent invalid entries  

---

## 🧠 Architecture

- Centralized `state` object (single source of truth)  
- Cached DOM elements for performance  
- Event delegation for dynamic UI updates  
- Clear separation of logic, rendering, and events  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (ES6+)
